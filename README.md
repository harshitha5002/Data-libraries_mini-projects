# Data Analysis Portfolio

This repository contains 10 Python projects demonstrating data analysis skills using Pandas, NumPy, and Matplotlib.
Each project includes analysis scripts, datasets, and visualizations.

## Projects Included

1. **Battery Performance Analysis** – Monitor battery health and usage trends.
2. **Air Quality Index (AQI) Analysis** – Analyze daily/monthly AQI trends.
3. **College Attendance Analytics** – Track student attendance and visualize trends.
4. **Electricity Consumption Analysis** – Analyze household electricity usage.
5. **Internet Usage Analysis** – Study internet usage patterns and peak times.
6. **Website Traffic Analysis** – Analyze website visits and identify peak days.
7. **Battery Performance Analysis** – Monitor battery health and usage trends.
8. **Traffic Flow Analysis** – Analyze traffic patterns and daily/monthly traffic counts.
9. **Public Transport Ticket Revenue Analysis** – Track daily and monthly ticket revenue.
10.**Website Traffic Analysis** – Analyze website visits and identify peak days.
